**📌 Overview**

This project presents an Android-based Home Appliance Service and Product Management Application designed to help users browse household appliances and request service support through a mobile interface.

The application provides a user-friendly platform for customers to explore appliances, manage accounts, and access service-related functionalities. The system integrates Android UI components with a local SQLite database to manage user authentication and application data.

Unlike traditional product browsing applications, this system combines product viewing, service request management, and customer interaction features within a single mobile platform, improving user convenience and accessibility.

The application demonstrates how mobile technology can simplify product discovery and service management for home appliances, making it easier for users to access product information and service assistance directly from their smartphones.

**🎯 Objectives**

Develop a mobile application for managing home appliance services

Provide secure user authentication using login and registration modules

Create an interactive interface for browsing appliance products

Integrate a local database system for storing user information

Enable easy access to appliance service features through a mobile platform

Improve user experience through responsive Android UI design

**📊 System Architecture**

The application follows a multi-activity Android architecture, where different activities handle specific tasks such as login, registration, product browsing, and service navigation.

A database helper class (DBHelper) manages database operations including user registration, authentication, and data storage.

The system ensures smooth communication between UI layouts, Java activity classes, and the SQLite database, enabling efficient data handling and application flow.
